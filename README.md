# Sporcle-E-Card

## Why I made this
This card was made for **_Sporcle Geography_** because of my love for it. I have learnt so much from playing the games and it has really expanded my knowledge of geography. I also enjoy playing with friends, which has brought us together.

## Frameworks/Languages Used
- Svelte
- HTML, JS, and CSS (within the Svelte app)

## Challenges Encountered
This project taught me a lot as I pushed myself to make it using features I had never tried before. I used Svelte, which was completely new to me, and tried the animations within this. My basic JS knowledge really helped with this, as I was able to implement functionality to the animations. I used tutorials online to help with this, as well as the example card, which I was able to expand upon to make it unique to my project.

## Sugar Rushes
### Learn a New Web-Framework

For this project I used Svelte, a framework I have never used before. I experimented with adding animations that Svelte allows for including sliding text, and colour changing. I was able to use the documentation and other people's tutorials to learn how to do this, which was really cool as I've never really worked with animations in websites.
